本地生活探店社交平台
项目简介
这是一套前后端分离的本地生活服务社交平台，基于 SpringBoot + Redis 开发，复刻真实互联网本地生活业务场景。
涵盖用户登录、商户门店浏览、探店笔记发布、点赞收藏、用户关注、同城推荐、限时优惠券秒杀等完整业务链路。项目重点落地Redis 高并发缓存设计、分布式锁、接口限流、全局唯一 ID、缓存穿透 / 击穿 / 雪崩解决方案等后端高频面试与工程实战知识点，适合作为 GitHub 开源项目、简历实战项目、后端进阶学习案例。
技术栈
后端
核心框架：SpringBoot、SpringMVC、MyBatis-Plus
缓存中间件：Redis、Redisson
数据库：MySQL
项目构建：Maven
工具依赖：Lombok、Knife4j 接口文档
前端
Vue2
Element UI
Axios
功能模块
用户模块手机号短信登录、登录状态保持、个人资料编辑
门店模块门店分类查询、门店详情展示、热门门店推荐、热点数据缓存优化
探店笔记模块发布探店图文、笔记点赞、热门笔记排行、点评互动
社交关注模块用户关注 / 取关、粉丝与关注列表、关注信息流推送
优惠券秒杀模块限时优惠券领取、高并发秒杀、分布式锁防超卖、库存安全扣减
同城模块基于地理位置的同城门店、同城笔记智能推荐
项目亮点
完整落地 Redis 缓存穿透、击穿、雪崩三种经典问题解决方案
基于 Redisson 实现可重入分布式锁，解决秒杀并发超卖问题
自研全局唯一 ID 生成器，替代数据库自增主键
利用 Redis 集合、哈希结构实现点赞去重与计数，性能远超数据库直查
关注信息流采用推拉结合设计，适配不同业务流量场景
秒杀接口实现限流、异步削峰，提升系统并发承载能力
热点数据缓存预热、过期策略优化，减少数据库压力
运行环境
JDK 1.8+
MySQL 5.7 / 8.0
Redis 6.0+
Maven 3.6+
快速部署
克隆项目至本地
导入项目提供的 MySQL 数据库脚本，初始化库表
修改 application.yml 配置文件，替换本地 MySQL、Redis 连接地址与账号密码
启动 SpringBoot 主程序
访问接口文档：http://localhost:8080/doc.html
启动前端 Vue 项目，配置代理即可完整联调使用
